
# CodeFrame
CodeFrame is a powerful and intuitive code snippet creation tool.
 


## Tech Stack

- React
- Redux-Toolkit
- Codemirror
## Features

- Gradient Selection
- Color Selection
- Dark Mode
- Line Number
- Font Size
- Padding 
- Reset
- Export you snippet
